# Security Operation Center L1: Проект-визитка

## 📖 О проекте
Данный веб-проект создан как наглядное пособие для студентов, изучающих информационную безопасность, и соискателей, желающих освоить профессию "Специалист по кибербезопасности (первая линия SOC)". 

Сайт разработан с учетом последних трендов веб-дизайна.

## 🚀 Как запустить локально
1. Клонируйте репозиторий: `git clone [ссылка на ваш репозиторий]`
2. Откройте файл `index.html` в любом современном браузере.
